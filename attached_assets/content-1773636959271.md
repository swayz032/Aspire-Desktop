### How to Import

Add this import to your component file

```jsx
1import ThreeDCarousel, { ThreeDCarouselItem } from "@/components/ThreeDCarousel";
```

### Props

Properties and options for customizing this component

| Name | Type | Default | Description | Required |
| --- | --- | --- | --- | --- |

|     |     |     |     |     |
| --- | --- | --- | --- | --- |
| items | ThreeDCarouselItem\[\] | - | An array of card items to display in the carousel | Required |
| autoRotate | boolean | true | Whether the carousel should auto-rotate | Optional |
| rotateInterval | number | 4000 | Interval (in milliseconds) between auto-rotations | Optional |
| cardHeight | number | 500 | Height of each carousel card in pixels | Optional |
| title | string | - | Optional section title | Optional |
| subtitle | string | - | Optional subtitle text for the section | Optional |
| tagline | string | - | Optional tagline or description below the title | Optional |
| isMobileSwipe | boolean | true | Enable swipe navigation on mobile | Optional |

### Basic Usage

Here’s a simple example of how to use this component

```jsx
1const items: ThreeDCarouselItem[] = [\
2  {\
3    id: 1,\
4    title: "6th SENSE Safety System",\
5    brand: "FireCat Group",\
6    description: "AI-driven smart uniform tech for law enforcement, military & firefighters.",\
7    tags: ["Safety", "Military", "AI Sensors", "Monitoring"],\
8    imageUrl: "/img/firecat.jpg",\
9    link: "/projects/firecat"\
10  }\
11];
12
13<ThreeDCarousel
14  items={items}
15  autoRotate={true}
16  rotateInterval={4000}
17  cardHeight={500}
18/>
```
