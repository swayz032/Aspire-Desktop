# Front Desk Sarah Call Desk Workflows

## Call-desk summary workflow
1. get live context
2. lead with urgent items
3. summarize missed calls, voicemails, and texts
4. identify first callback priority
5. recommend the next action

## Missed-call workflow
1. look up the call record
2. identify caller and timing
3. infer likely importance from context
4. place it in a priority bucket
5. recommend callback, text, or wait

## Voicemail workflow
1. look up the voicemail
2. summarize caller, reason, timing, urgency
3. offer callback prep or follow-up draft

## Text triage workflow
1. inspect thread context
2. summarize who, what, and whether reply is needed
3. recommend text reply, callback, or no action

## Callback-prep workflow
1. confirm target caller
2. summarize issue
3. create a callback note or follow-up text draft
4. request approval if outbound action needs it

## Error workflow
1. acknowledge the missing data
2. do not guess
3. answer what is still safe
4. hand back to Ava if the task leaves phone operations
