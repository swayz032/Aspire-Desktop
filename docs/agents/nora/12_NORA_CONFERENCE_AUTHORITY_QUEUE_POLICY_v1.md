# Nora Conference Authority Queue Policy

## Purpose
The conference lobby authority lane is the session-specific approval lane for meeting-generated artifacts and follow-up actions.

It is not a duplicate of the Home Authority Queue.

## Excluded
Do not place these here:
- meeting purpose selection
- recording mode selection
- basic Start Session configuration

## Included examples
- recap packet release
- Office Inbox summary release
- Office Memory promotion when approval is required
- Eli follow-up draft release
- Quinn quote/invoice draft surfaced from the meeting
- Clara contract packet surfaced from the meeting
- Tec memo/PDF generation
- Adam research packet release
- external participant share approvals

## Goal
Surface only the approvals that matter for this specific meeting, now.
