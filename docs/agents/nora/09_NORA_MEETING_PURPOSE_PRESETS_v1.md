# Nora Meeting Purpose Presets

## Source of truth
Meeting purpose is selected in the Start Session modal and passed to Nora as runtime context.

## Purpose effects
Purpose should change:
- briefing behavior
- recap emphasis
- specialist trigger map
- Office Inbox summary style
- Office Memory tags
- conference authority lane suggestions

## Internal
- Focus: decisions, owners, blockers, next actions
- Likely specialists: Finn, Front Desk Sarah, Tec, Adam
- Office Memory tags: `purpose=internal`

## Client
- Focus: commitments, deliverables, expectations, next step
- Likely specialists: Eli, Quinn, Clara, Adam
- Office Memory tags: `purpose=client`

## Vendor
- Focus: pricing, terms, timing, dependencies
- Likely specialists: Adam, Finn, Clara, Quinn
- Office Memory tags: `purpose=vendor`

## Deal
- Focus: objections, urgency, blockers, quote/contract triggers
- Likely specialists: Eli, Quinn, Clara, Adam, Tec
- Office Memory tags: `purpose=deal`

## Network
- Focus: relationship notes, follow-up opportunities
- Likely specialists: Eli, Adam, Tec
- Office Memory tags: `purpose=network`
