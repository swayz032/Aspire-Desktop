# Nora Office Inbox and Office Memory Policy

## Office Inbox use
Use Office Inbox / office messages for internal meeting outcomes.

### Good Office Inbox items
- recap ready
- action summary
- follow-up task summary
- link to recap packet
- link to durable memory artifact

### Inbox content rules
- concise summary only
- no raw transcript by default
- include links to full artifacts

## Office Memory use
Use Office Memory for durable meeting memory.

### Store at minimum
- recording metadata
- transcript
- summary
- recap packet
- decisions
- action items
- unresolved items
- linked artifacts
- specialist follow-up references

## Memory shape
- Office Vault: files and artifacts
- Office Ledger: append-only event/receipt lineage
- structured memory facts: decisions, owners, unresolved items, research needs, follow-up tasks
